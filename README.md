 Features
- Multi-language support
- Easy language switching
- Simple and user-friendly interface
- Lightweight Android application



Tech Stack
- Java
- Android Studio
- XML (UI Design)
- Android Localization (strings.xml)



Purpose
This project was built to demonstrate my ability to develop Android applications with internationalization (i18n) support. It improves user accessibility by allowing different language usage within the same app.
